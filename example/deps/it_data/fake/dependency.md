# it_data/fakeの依存関係

## \*/eをパッケージとみなさない場合
* a -> b -> c
* a -> d
* d -> e

## \*/eをパッケージとみなす場合
* a -> b/e
* a -> d/e
* b-> c/e
* c-> a/e
* c-> c/e
* d -> e/e
* e -> d/e

上記の通り、サイクリックな依存関係はなくなる。
当然これは意図したものではないので、「\*/eをパッケージとみなさない場合」を採用すべきだが、

* \*/eはパッケージではなく
* [a-e]/はパッケージである

のは、ファイル構造から読み解くことはできない。
